.. _aws.codebuild.filters.json-diff:

json-diff
+++++++++
Compute the diff from the current resource to a previous version.

A resource matches the filter if a diff exists between the current
resource and the selected revision.

Utilizes config as a resource revision database.

Revisions can be selected by date, against the previous version, and
against a locked version (requires use of is-locked filter).

  .. raw:: html

  .. code-block:: yaml

    properties:
      selector:
        enum:
        - previous
        - date
        - locked
      selector_value:
        type: string
      type:
        enum:
        - json-diff
    required:
    - type

.. _aws.codebuild.actions.delete:

delete
++++++
Action to delete code build

It is recommended to use a filter to avoid unwanted deletion of builds

:example:

.. code-block:: yaml

        policies:
          - name: codebuild-delete
            resource: codebuild
            actions:
              - delete
