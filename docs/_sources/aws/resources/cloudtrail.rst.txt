event-selectors
+++++++++++++++
Filter a cloudtrail by its related Event Selectors.

      filters:
      - type: event-selectors
        key: EventSelectors[].IncludeManagementEvents
        op: contains

.. _aws.cloudtrail.filters.is-shadow:

is-shadow
+++++++++
Identify shadow trails (secondary copies), shadow trails
can't be modified directly, the origin trail needs to be modified.

Shadow trails are created for multi-region trails as well for
organizational trails.

.. _aws.cloudtrail.filters.json-diff:

json-diff
+++++++++
Compute the diff from the current resource to a previous version.

A resource matches the filter if a diff exists between the current
resource and the selected revision.

Utilizes config as a resource revision database.

Revisions can be selected by date, against the previous version, and
against a locked version (requires use of is-locked filter).

.. container:: toggle

  

  .. raw:: html

      selector:
        enum:
        - previous

        - locked

        type: string
      type:
        enum:
        - json-diff
    required:
    - type

update-trail
++++++++++++
Update trail attributes.

:Example:

.. code-block:: yaml

   policies:
     - name: cloudtrail-set-log
       resource: aws.cloudtrail
       filters:
        - or:
          - KmsKeyId: empty
          - LogFileValidationEnabled: false
       actions:
        - type: update-trail
          attributes:
            KmsKeyId: arn:aws:kms:us-west-2:111122223333:key/1234abcd-12ab-34cd-56ef
