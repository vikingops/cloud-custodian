.. _aws.dms-endpoint.actions.delete:

delete
++++++
Delete a DMS endpoint

:example:

.. code-block:: yaml

      policies:
        - name: dms-endpoint-no-ssl-delete
          resource: dms-endpoint
          filters:
            - EngineName: mariadb
            - SslMode: none
          actions:
            - delete

.. _aws.dms-endpoint.actions.modify-endpoint:

modify-endpoint
+++++++++++++++
Modify the attributes of a DMS endpoint

:example:

.. code-block:: yaml

      policies:
        - name: dms-endpoint-modify
          resource: dms-endpoint
          filters:
            - EngineName: sqlserver
            - SslMode: none
          actions:
            - type: modify-endpoint
              SslMode: require
