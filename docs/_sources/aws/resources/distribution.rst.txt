.. _aws.distribution.filters.json-diff:

json-diff
+++++++++
Compute the diff from the current resource to a previous version.

A resource matches the filter if a diff exists between the current
resource and the selected revision.

Utilizes config as a resource revision database.

Revisions can be selected by date, against the previous version, and
against a locked version (requires use of is-locked filter).

.. container:: toggle

  

  .. raw:: html

  .. code-block:: yaml

    properties:
      selector:
        enum:
        - previous
        - date
        - locked
      selector_value:
        type: string
      type:
        enum:
        - json-diff
    required:
    - type

.. _aws.distribution.actions.set-attributes:

set-attributes
++++++++++++++
Action to update the attributes of a distribution

:example:

.. code-block:: yaml

    policies:
    - name: enforce-distribution-logging
      resource: distribution
      filters:
        - type: value
          key: "Logging.Enabled"
          value: null
      actions:
        - type: set-attributes
          attributes:
            Comment: ""
            Enabled: true
            Logging:

                IncludeCookies: false

.. _aws.distribution.actions.set-protocols:

set-protocols
+++++++++++++
Action to set mandatory https-only on a Distribution

:example:

.. code-block:: yaml

        policies:
          - name: distribution-set-ssl
            resource: distribution
            filters:
              - type: value
                key: CacheBehaviors.Items[].ViewerProtocolPolicy
                value: allow-all
                op: contains
            actions:
              - type: set-protocols
                ViewerProtocolPolicy: https-only
