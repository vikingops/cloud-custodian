.. _aws.catalog-portfolio.filters.json-diff:

json-diff
+++++++++
Compute the diff from the current resource to a previous version.

A resource matches the filter if a diff exists between the current
resource and the selected revision.

Utilizes config as a resource revision database.

Revisions can be selected by date, against the previous version, and
against a locked version (requires use of is-locked filter).

.. _aws.catalog-portfolio.actions.remove-shared-accounts:

remove-shared-accounts
++++++++++++++++++++++
Action to delete Portfolio share with other accounts

:example:

.. code-block:: yaml

        policies:
          - name: catalog-portfolio-delete-share
            resource: aws.catalog-portfolio
            filters:
              - type: cross-account
            actions:
              - type: remove-shared-accounts
                accounts: matched

          - name: catalog-portfolio-delete-shared-account
            resource: aws.catalog-portfolio
            filters:
              - type: cross-account
            actions:
              - type: remove-shared-accounts
                accounts: ['123456789123']
