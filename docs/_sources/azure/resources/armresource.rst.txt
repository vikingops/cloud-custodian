azure.armresource
=================

Azure Arm Resource

:example:

This policy will find all ARM resources with the tag 'Tag1' present

.. code-block:: yaml

    policies
      - name: find-resources-with-Tag1
        resource: azure.armresource
        filters:
          - tag:Tag1: present
